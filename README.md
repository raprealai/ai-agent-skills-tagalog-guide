## Ano ba Talaga ang Agent Skills?
 
### Simple Explanation (Tagalog)
 
**Agent skills** ay parang **"cheat sheets" o "recipe cards"** na binibigay mo sa iyong AI assistant para alam niya kung paano mag-handle ng specific tasks mo.
 
Halimbawa:
- Kung palagi kang nag-rereply ng emails sa customers...
- Gumawa ka ng **"Customer Email Skill"**
- Pag may email na, automatic na alam ng AI ang tone, format, at style mo!
 
### Technical Definition (English)
 
Agent skills are **folders or modules** containing:
- рҹ“қ Instructions and metadata
- рҹӣ пёҸ Optional scripts or templates
- рҹҺҜ Specific workflows or domain knowledge
 
They allow AI agents to:
- Discover capabilities on-demand
- Load relevant context only when needed
- Execute specialized tasks consistently
 
**Think of it as:** Plugins, power-ups, or playbooks for your AI agent.
